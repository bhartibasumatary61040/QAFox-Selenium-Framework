# QAFox Selenium Automation Framework

A structured Selenium Test Automation Framework built using Java, Selenium WebDriver, TestNG, and Maven following the Page Object Model (POM) design pattern.

This project demonstrates real-world UI automation practices and scalable framework architecture used in QA Automation roles.

---

## Tech Stack

- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- Page Object Model (POM)  
- ChromeDriver  

---

## Project Structure

QAFox-Selenium-Framework

src/main/java  
  base        - Base setup & driver initialization  
  pages       - Page Object Model classes  
  utils       - Utility/helper classes  

src/test/java  
  tests       - TestNG test classes  

testng.xml    - Test suite configuration  
pom.xml       - Maven dependencies  
test-output   - TestNG reports

## How to Run the Project

1. Clone the repository  
2. Import as Maven project in Eclipse/IntelliJ  
3. Run testng.xml file  
4. Open test-output/index.html to view report
